What is Homecooked?

Homecooked is a ASGI-compliant web framework similar to Flask and FastAPI. It includes support for dynamic routing of paths and middleware, and also features a static server. Through PyDantic, Homecooked is able to deserialize JSON requests to Python objects and via the Meal Template Engine, Homecooked supports it's own, very special, templating language. It also has support for SubRouters, which is Homecooked's equivalent to Blueprints in Flask.

What is Meal?

Meal is a templating language for the Homecooked ASGI framework. It is similar to Jinja2 and other templating languages, except all terminals are replaced with emojis.

How to get started with Meal and Chef?

To test out Meal, this site includes Chef, an online IDE to test out Meal templates. You can write your own Meal templates and provide context to render the templates. Chef also includes a preview of the rendered template.
