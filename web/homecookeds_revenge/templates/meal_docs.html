🥢 title 🥢

Meal is a templating language for the Homecooked ASGI framework. It is similar to Jinja2 and other templating languages, except all terminals are replaced with emojis.

How to get started with Meal and Chef?

To get started with Meal and Chef, read the Meal Docs and try out the Chef Creator.

Writing your first Meal Program

In Meal, if it's not a variable, string or number, it's an emoji. Meal is a very powerful templating language, allowing for control structures, loops, and more.

In the below program, we will do some basic math and print the result. 

🧆🍴 a 🍇 5 🍴 

🍴 b 🍇 10 🍴 

🥢 a 🍦 b 🥢🧆

This program will output 15 to the screen.

  • 🧆🍴🧆 denotes the start of a statement (or block scope) and end of a statement (or block scope)
  • 🧆🍇🧆 represents equality. So a = 5
  • 🧆🍦🧆 is the addition operator. So a + b = 15, as b = 10
  • 🧆🥢🧆 denotes an expression, which outputs text to the screen

Control Flow in Meal

Meal supports control flow structures such as if-else statements and loops.

In the below program, we will check if a is greater than b and print the result if it is. While it is not, we will increase the value of a 

🧆🍴 a 🍇 5 🍴

🍴 b 🍇 10 🍴

🍴 🌯 a 🍰 b 🥄

    🍴 a 🍇 a 🍦 1 🍴

🥄🍴
🥢 a  🥢
🧆

This program will output 10 to the screen, as a is incremented 5 times before it is equal to b.

  • 🧆🍴🧆 denotes the start of a statement
  • 🧆🥄🧆 denotes control flow scoping
  • 🧆🍇🧆 represents equality. So a = 5
  • 🧆🍰🧆 is the less than operator. So a < b
  • 🧆🌯🧆 denotes a while statement. So while a < b, execute the loop

    • In the next program, we demonstrate a for loop. We will loop through a range of numbers and print the result. 

    🧆
🍴 a🍇 🍎 1🌭2🌭5🌭21🌭123🍏🍴
    
🍴 🍔 num 🍟 a 🥄
    
    🍴 🍕 num 🥧 2 🥄
    
        num = 🥢 num  🥢
    
    🥄
    
    🥙 num 🥧 21 🥄
    
        num is now 21, see 🥢 num  🥢
     🥄🍴
🥄🍴
🧆

    This program will iterate through the list [1, 2, 5, 21, 123]. When the value of 'num' is equal to 2 and 21, the program will output to the screen.

    • 🧆🦀🧆 is the start of an array
    • 🧆🌭🧆 is the item delimter of an array
    • 🧆🦞🧆 is the end of an array
    • 🧆🍔🧆 is equivalent to 'for'
    • 🧆🍟🧆 is equivalent to 'in'
    • 🧆🍕🧆 is equivalent to 'if'
    • 🧆🥙🧆 is equivalent to 'else if'

    In more readable psuedocode, the program is equivalent to: 

    a = [1,2,5,21,123] 
    

    
for num in a:
    
    if num == 2:
    
        print(num)
    
    else if num == 21:
    
        print(num)

Meal context

As Meal is made to be used with web servers, and we often want to render dynamic information from the webserver, Meal supports providing a 'render context' to the Meal template. When returning a TemplateResponse, the context to provide to Meal is the second argument, and consists of a dict mapping variable names to values.

In the below program, we will render a template with a context. Suppose we provide the following context: 

context = {'name': 'Meal Docs', 'author': 'Homecooked', 'value': 5}

And the following template: 

🧆
The name is: 🥢name🥢

The author is: 🥢author🥢

Value: 🥢max(value 🍧 10🌭2)🥢
🧆

🧆🥢name🥢🧆 will be replaced by 'Meal Docs', 🧆🥢author🥢🧆will be replaced by 'Homecooked', and the output for value will be the max of 'value - 10' and 2.
